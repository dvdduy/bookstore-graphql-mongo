# bookstore-graphql-mongo
This is to sample project to develop a Angular single page web application with GraphQL api.
- GraphQL api: Asp.Net Core (Net 5), HotChocolate (Annotation-based), MongoDB. 
- Angular 12: use Apollo Angular to fetch GraphQL data. Also, concept of smart and presentation components are used. The AppComponent plays role of smart component and other components are presentation ones.
